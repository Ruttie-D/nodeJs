API Documentation

This documentation provides an overview of the available API endpoints for managing users and cards.

Users API

  • GET /users: Retrieves a list of all users (admin access required).
  • GET /users/:id: Retrieves details of a specific user.
  • POST /users/register: Registers a new user.
  • POST /users/login: Authenticates an existing user.
  • PUT /users/:id: Updates user details.
  • DELETE /users/:id: Removes a user account.

Cards API

  • GET /cards: Retrieves a list of all available cards.
  • GET /cards/:id: Retrieves details of a specific card.
  • POST /cards: Creates a new card (restricted to business users).
  • PUT /cards/:id: Updates an existing card.
  • DELETE /cards/:id: Deletes a card.
  • PATCH /cards/:id: Toggles the like status of a card.
